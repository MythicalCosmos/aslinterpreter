ASL Interpreter with Audio Transcription

A real-time American Sign Language (ASL) interpreter built in Python using computer vision, machine learning, and speech processing. This project can recognize ASL hand signs through a webcam feed and includes audio transcription features for converting spoken language into text.

Features
Real-time ASL hand sign recognition
Webcam-based gesture detection
Audio-to-text transcription
Live subtitle generation
Machine learning powered gesture classification
Multi-threaded processing for smooth performance
Easy-to-expand gesture dataset
Cross-platform support
Technologies Used
Python 3.10+
OpenCV
MediaPipe
TensorFlow / PyTorch
SpeechRecognition
PyAudio
NumPy
scikit-learn
